# Проект: Автотесты API для Stellar Burgers
Описание проекта  
Этот проект содержит автоматизированные тесты для API сервиса Stellar Burgers. Тесты покрывают основные сценарии работы с API, включая создание пользователей, авторизацию и управление заказами.

## Технологии
    * Java 11
    * Maven 
    * JUnit 4
    * REST Assured
    * Allure Framework

### Поддерживаемые тестовые сценарии
#### 1. Создание пользователя
- Создание уникального пользователя
- Попытка создания уже зарегистрированного пользователя
- Создание пользователя без обязательного поля

#### 2. Авторизация пользователя
- Вход под существующими учетными данными
- Попытка входа с неверными учетными данными
- Попытка входа без email

#### 3. Управление заказами
- Создание заказа с авторизацией
- Создание заказа без авторизации
- Создание заказа с ингредиентами
- Создание заказа без ингредиентов
- Попытка создания заказа с неверным хешем ингредиентов
